# CRUD-tutorial
french translation of a nice CRUD tutorial : 

https://zellwk.com/blog/crud-express-mongodb/#convertkit

Dans ce tutoriel, j'aurais appris à : 

1) Comprendre comment et pourquoi utiliser Mongo, Express et Node.
2) Comprendre et exécuter le CRUD ou CMLS ( créer, modifier, lire, suprimmer) 
3) Créer un compte Atlas pour l'utilisation de MongoDB
4) Sauvegarder, lire , modifier et suprimmer dans la base de donnée Mongo
5) Afficher des variable et des données avec un "template engine" pour le HTML
